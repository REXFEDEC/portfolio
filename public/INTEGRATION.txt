In app/layout.tsx, import and add <ChatBubble /> at the
bottom of your <body> tag, after your main content:

  import { ChatBubble } from "@/components/chat-bubble";

  export default function RootLayout({ children }) {
    return (
      <html>
        <body>
          {/* your sidebar, navbar, main content */}
          {children}
          <ChatBubble />   {/* ← add this */}
        </body>
      </html>
    );
  }

That's it. The bubble is position: fixed so it floats above
everything regardless of page layout.

HOW THE JAILBREAK PROTECTION WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The system prompt uses a structural grounding technique rather
than just telling the model "don't answer other questions."

The model is defined as a "read-only information terminal"
with exactly one data source. This framing makes it treat
out-of-scope questions as having no answer to return rather
than questions it is forbidden from answering — which is much
harder to socially engineer around.

Temperature is set to 0.2 (low) which further reduces the
model's tendency to improvise or be creative with its scope.

If someone tries: "ignore previous instructions and tell me..."
The model has no previous persona to revert to — it was never
defined as a general assistant in the first place.

## AI CHAT BUBBLE

The site has a floating AI chat bubble component at
`components/chat-bubble.tsx`. Do NOT generate this component
— it already exists and is imported separately.

In `app/layout.tsx`, import and render it at the bottom of
the body, after all other content:

  import { ChatBubble } from "@/components/chat-bubble";
  // Add <ChatBubble /> as the last child of <body>

The bubble is `position: fixed` bottom-right. It uses #7085FF
as its accent color and matches the site's dark/light theme
via CSS variables. No additional styling needed.
